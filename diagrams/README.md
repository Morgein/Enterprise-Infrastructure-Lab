# Diagrams and Screenshots

Screenshots are stored under:

```text
diagrams/windows-server/
```

Current numbering convention:

```text
01-16  AD, DNS, DHCP and domain client
17-31  OU, GPO, SMB share and mapped drive
32-40  iSCSI storage
41-53  Veeam backup and restore
```

The screenshots are intentionally kept in one folder for now to avoid breaking existing links.  
They can be split into subfolders later after all documentation links are updated.
